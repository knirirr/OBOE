The WorldClim – Global Climate Data website (http://www.worldclim.org/) provides free climate data for academic and other non-commercial use1. OBOE-CDE provides, with permission from WorldClim, a simple data extraction service for any user-specified sub-region of the WorldClim global coverage. The output data are written in GeoTiff format files that are readily reusable for further analysis, e.g. climate spatial pattern analysis, or species distribution modelling.

OBOE-CDE is currently available for:

·         WorldClim Version 1.4, release 3; Current conditions, ~1950-2000.

In this document you will find maps and histograms for all 68 variables. The .zip file contains the data as extracted from the WorldClim global dataset. You will also find files for all the figures in this document for possible reuse in publications.

·         /output/data/ contains all the data files.

·         /output/images/figures/ contains all the figures.
